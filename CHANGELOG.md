# Changelog

All notable changes to the NeoCaptcha Server SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-03

### Added
- Initial release of the NeoCaptcha Server SDK
- `verifyToken` function for verifying captcha tokens
- `VerifyTokenOptions` and `VerifyTokenResult` TypeScript interfaces
- Full TypeScript support with type definitions
- Express.js and Next.js integration examples
